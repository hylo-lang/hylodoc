type Vector2

Represents a 2D vector as a pair of two floating point numbers

Declaration 
type Vector2 = {x: Float64, y: Float64}

Details

This initializer allows you to create a new instance of the type by directly copying the bit pattern from another UInt32 value. This can be useful for low-level operations where you need to manipulate the raw binary representation of numbers.
